## 📖 عن اللعبة

لعبة تفاعلية ممتعة تختبر قدرتك على تخمين الأمثال العربية الشعبية والحكم المشهورة من خلال الرموز التعبيرية (الإيموجي). تتضمن اللعبة مجموعة متنوعة من الأسئلة بمستويات صعوبة مختلفة.

## ✨ المميزات

- 🎯 **60+ سؤال متنوع** من الأمثال والحكم العربية
- 📊 **3 مستويات صعوبة**: سهل، متوسط، صعب
- ❤️ **نظام الحياة**: 3 فرص للخطأ
- 🏆 **الرقم القياسي**: يُحفظ تلقائياً في المتصفح
- 💡 **3 تلميحات** لكل سؤال
- 🔀 **ترتيب عشوائي** للرموز في كل سؤال
- 🔊 **مؤثرات صوتية** قابلة للتشغيل/الإيقاف
- 📱 **تصميم متجاوب** يعمل على الجوال والكمبيوتر
- 📤 **مشاركة النتيجة** مع الأصدقاء
- 🎊 **تأثيرات بصرية** عند تحقيق رقم قياسي

## 🚀 التشغيل

### الطريقة الأولى: مباشرة
افتح ملف `index.html` في أي متصفح حديث.

### الطريقة الثانية: خادم محلي
```bash
# باستخدام Python
python -m http.server 8000

# أو باستخدام Node.js
npx serve
```
ثم افتح `http://localhost:8000` في المتصفح.

## 📁 هيكل المشروع

```
emoji-charades/

```

## 🎵 الملفات الصوتية

اللعبة تعمل بدون ملفات صوتية، لكن لتجربة أفضل يُنصح بإضافة ملفات MP3 قصيرة في مجلد `assets/sounds/`.

### مواصفات الملفات الصوتية المقترحة:

| الملف | الوصف | المدة المقترحة |
|-------|-------|----------------|
| `correct.mp3` | صوت نجاح/تصفيق قصير | 0.5-1 ثانية |
| `wrong.mp3` | صوت خطأ/تنبيه | 0.5 ثانية |
| `hint.mp3` | صوت تنبيه خفيف | 0.3 ثانية |
| `levelup.mp3` | صوت احتفالي | 1-2 ثانية |
| `gameover.mp3` | صوت حزين/درامي | 1-2 ثانية |
| `highscore.mp3` | صوت فوز/إنجاز | 2-3 ثانية |
| `click.mp3` | صوت نقرة زر | 0.1-0.2 ثانية |
| `start.mp3` | صوت بداية اللعبة | 1 ثانية |

## 🎮 طريقة اللعب

1. **اختر مستوى الصعوبة** من الشاشة الرئيسية
2. **اضغط زر التشغيل** لبدء اللعبة
3. **شاهد الرموز التعبيرية** وحاول تخمين العبارة
4. **اكتب إجابتك** في الحقل المخصص
5. **استخدم التلميحات** إذا احتجت مساعدة (تخصم من النقاط)
6. **اجمع أكبر عدد من النقاط** قبل نفاد الحياة!

## 📊 نظام النقاط

| المستوى | النقاط الأساسية | خصم التلميح |
|---------|-----------------|-------------|
| سهل | 10 نقاط | -3 لكل تلميح |
| متوسط | 20 نقطة | -3 لكل تلميح |
| صعب | 30 نقطة | -3 لكل تلميح |

- **الحد الأدنى**: 5 نقاط لكل إجابة صحيحة
- **انتقال المستوى**: كل 5 إجابات صحيحة
